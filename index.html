Pipeline Architecture

Phase 1

Post-Scrape Translation

Raw Chinese product data from 1688.com is localised to UK English using Lingo.dev's AI engine with a custom B2B glossary.

Phase 2

Semantic Bridge

English procurement intent is transformed into native Chinese search bundles — primary query + synonyms + negative keywords — before scraping begins.

Phase 3

Validation Layer

Confidence scoring with Jaro-Winkler fuzzy matching, blacklist filtering, duplicate detection, and adaptive synonym expansion when confidence is low.

Phase 4

Vision Validation

GPT-4V image analysis cross-checks product images against the listing text, catching "bait and switch" listings that pass text filtering.
